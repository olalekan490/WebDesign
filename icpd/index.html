This document provides a comprehensive architectural overview of the proposed system. It is intended to capture and convey the significant and design decisions that have been made for this project. On the other hand, the need for various client device has been taking into consideration, in this regards the application presents UI(User interface) base on the device screen size of the user that access it. it can be access through Mobile, Personal PC and other devices with respective view taken into an account. Therefore the application is very much responsive and simple to use.

Customer Data Manager : This Module enables you to efficiently manage customer data, this is where various operations on the customer can be perform. However, the major operation allows are Create, Retrieve, Update and Delete. In contrast only user with right access will be able to trigger those operations that are allowed by the system.
